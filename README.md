# factor-cli-hello-world

## Description
A POC for factor programming language.

## Tech stack
- g++

## Docker stack
- ubuntu:22.04
- rzuckerm/factor

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
